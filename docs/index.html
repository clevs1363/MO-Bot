About the Bot

A fun passion project for the boys. It started as a replacement to Groovy and Rhythm, and has evolved into a collection of random commands for the server. Meant to make repeated tasks easier, and to get a chuckle from time to time.. I'm always looking for cool things to develop with Obotma, so don't be afraid to !request stuff. Our screening process is pretty loose.

See the links below to see features in progress and the Future of Obotma.

Commands


General format: !command [argument]. Command should be prefixed with !, but the argument should not have the brackets [ ] unless specified. Each command indicates whether the argument is required or not.


Music

!play [url] : plays YouTube video from given url. Must be in a voice chat | Argument required? yes
!play [query] : searches and plays YouTube audio with given query. Must be in a voice chat | Argument required? yes
!pause: pauses current song
!resume: resumes song
!skip: skips to next song in the queue if there is one queued
!join: have a friend join you in voice chat

Requests

!request [text] : add a feature request to a queue, up to a max of 15. Use without arguments to show the numbered list of requests | Argument required? no
!delete_request [number] : deletes request at position [number] in the queue | Argument required? yes

Memes

!pun: sends a random pun
!lenny [number] : sends [number] of random lennies. Defaults to 1 if number not provided. | Argument required? no
!meme: summons a random meme. Quality not guaranteed.

Dice Roller

!r XdY+Z: rolls X number of a dY and adds a modifer Z. Example: !r 3d6+5
!r stats: rolls character stats using 4d6 drop lowest method

Polls

Formatting for this one is very important.
!poll {title} [Option 1] [Option 2] [Option 3] ... : creates a poll for the given options. Supports up to 9 options. Braces and brackets should be included | Argument required? yes

Risk of Rain 2

!swag [time] : creates a swag session and tags the @ror2 role to ask to swag at given time. Max of 4 swaggers. Clears previous swag ses. | Argument required? yes
  • Time must be formatted appropriately. Accepts either military HH:MM time (ex: 17:50) or standard HH:MMxm time (ex: 12:50pm)
  • Now caps at 4 swaggers
  • Will notify current swaggers of the ses 10 minutes beforehand.
!swaggers: shows list of people current interested in swagging
!random_swag [number] : randomizes characters for each current swagger and randomly picks [number] amount of artifacts. Default 0 artifacts if not provided. | Argument required? no
!artifact [text] : returns description for artifact name given in [text]. Should be just "Chaos" or "Vengeance", not "Artifact of [text]" | Argument required? yes
!swag_report win OR loss : ends the current swag ses and clears the swagger list. Option (win/loss) to keep track of stats. | Argument required? no

Positivity

!nature [query] : fetches image related to query | Argument required? yes
!hug: send hugs
!encourage: sends affirmation, and an optional inspirational quote
!compliment: sends a random compliment

--NEW-- Emotes

!e [name] : sends emoji with given name. Use to send animated emojis. | Argument required? yes
!e [name] : when you use this command to react to a message, it will react to the targeted message with the given emote. | Argument required? yes

Scheduled Tasks

Daily message: sends a good-morning message with Today In History (events, births, and deaths), quote of the day, random word of the day, and today's Inktober prompt.

Miscellaneous

!ses: gives the time, date, and location of the next ses
!abyses: same as previous but for Drew's campaign
!help: Sends a link to this site
!inktober: says today's Inktober prompt. Will also be sent with daily message

Admin Center

These functions require administrator permissions, or to be named CerealGuy69.
!(aby)ses [text] : changes next ses info and puts it in the current channel banner
!delete_request 0: clears request list
!scan: gives emoji stats in current channel. WARNING: will take a while to run, and will return a lot of data!
